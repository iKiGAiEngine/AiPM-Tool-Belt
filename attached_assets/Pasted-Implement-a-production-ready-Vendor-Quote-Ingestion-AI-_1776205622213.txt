1.4 Confidence gating
Use these thresholds:

HIGH_CONFIDENCE >= 0.85

MEDIUM_CONFIDENCE 0.65–0.84

LOW_CONFIDENCE < 0.65

Rules:

If average confidence < 0.65 OR any required fields missing (item description + unit cost), set status needs_review.

If OCR/AI fails, set status failed and offer manual review directly.

If user clicks “Reject Parse”, set status needs_review.

5) AI extraction contract
Create a single extraction service interface:

type ExtractionInput = {
  fileUrl: string;
  regions?: Array<{ page: number; x: number; y: number; w: number; h: number; label: string }>;
};

type ExtractedLineItem = {
  description: string;
  quantity?: number;
  uom?: string;
  unitCost?: number;
  extendedCost?: number;
  partNumber?: string;
  manufacturer?: string;
  confidence: number;
  sourcePage?: number;
  sourceBbox?: { x: number; y: number; w: number; h: number };
};

type ExtractionOutput = {
  vendorName?: string;
  quoteNumber?: string;
  quoteDate?: string;
  subtotal?: number;
  tax?: number;
  total?: number;
  currency?: string;
  lineItems: ExtractedLineItem[];
  avgConfidence: number;
  requiredFieldsMissing: string[];
};

If AI parse returns empty line items: status needs_review.
