	∙	What database is being used and where is the connection config

Step 2: Find violations
Search every file for these problems. Be specific — give me the file path and line number for every finding.
Database queries:
	∙	Any query that reads or writes project-scoped data without filtering by project_id
	∙	Any JOIN or aggregation that could pull records from multiple projects
	∙	Any SELECT * on a project-scoped table without a project filter
	∙	Missing RLS policies (if Supabase)
	∙	Missing indexes on project_id columns
API routes:

	∙	Any endpoint that trusts project_id from the client without server-side validation

Frontend:
	∙	Any API call that doesn’t pass project_id

AI agents:

File storage:

	∙	File access not checking project ownership
